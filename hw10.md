# 2

list={60,65,75,80,90,95}.               

## pseudocode:          

READ x              

FOR i from 0 to list.length-1           

IF x < list.i        

insert x before list.i              

ELSE            

insert x in the end of the list            

ENDIF               

ENDFOR              

{50,60,65,75,80,90,95}              

{60,65,67,75,80,90,95}              

{60,65,75,80,90,95,99}              

为了防止程序出错，开头与结尾是特殊情况，中间是一般情况。
